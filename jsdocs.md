# JSdocs

## install 
run the following commands:
``` 
npm install
npm install -g jsdoc --save-dev
```

add to package.json:
``` 
"scripts": {
    "docs": "jsdoc {path}.js"
  }, 
``` 

## usage
``` 
npm run docs
```


## access docs
locally run index.html located in /out/index.html
